# playcode

Experimental Playground for a VueUse sandbox

代码测试运行环境，utools 插件的形式

# 目标
与 npm-helper 插件打通， npm 库直接测试运行、node 环境运行
typescript 直接运行（运行 ts 体操测试）


# 目标
1. html + scss （类似 jsbin.com）
2. typescript
3. vue
4. react
5. node
6. deno
7. angular
8. rust
